Os principais comandos para criar um site

Comando 1: Criar Estrutura HTML

O comando <!DOCTYPE html> é utilizado para definir que o documento é um arquivo HTML5. Ele deve ser a primeira linha de qualquer documento HTML.

Comando 2: Estilizando com CSS

O comando link rel="stylesheet" href="styles.css" é usado para vincular um arquivo CSS ao seu documento HTML, permitindo que você estilize sua página.

### Código CSS (styles.css) css body { font-family: Arial, sans-serif; } h1 { font-size: 30px; color: blue; } h2 { font-size: 20px; color: red; } p { font-size: 15px; font-family: "Times New Roman", Times, serif; color: black; }
